# Embedded Test Framework in C

## 📌 Overview
This project demonstrates a basic embedded software test framework in C, inspired by safety-critical development practices.

## ⚙️ Features
- Unit testing in C
- Test automation via Makefile
- Structured test cases (nominal, boundary, robustness)

## 🧪 Example
Tested module: Calculator

## 🚀 How to Run
make
./run_tests

See /docs for requirements and test strategy
